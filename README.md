# Dostaffkin

## Overview
A web application for ordering and tracking deliveries. Users can place orders, track their status, and interact with delivery services through a simple interface.

## Features
- Place new delivery orders
- Track order status in real time
- View order history
- Responsive UI with navigation header

## Tech Stack
- **Framework:** Angular
- **Language:** TypeScript
- **Styles:** CSS
- **Build Tool:** Angular CLI

## Project Structure
- `src/app/` — main application code
  - `pages/` — feature pages: Home, Order, Track
  - `services/` — API integration
  - `header/` — navigation component
- `public/`, `docs/` — static assets

## Installation & Usage
```sh
npm install
npm start
```

## Scripts
- `npm start` — run development server
- `npm run build` — build for production
